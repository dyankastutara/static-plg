# Plasgos Front-End

## To Use
```javascript
- clone this repo
- cd plasgos-front
- yarn install
- yarn start
```
## Endpoint
| No. | Route                          | Page          |
|-----|--------------------------------|---------------|
| 1.  | http://localhost:3000/         | Home Page     |
| 2.  | http://localhost:3000/search   | Search Page   |
| 3.  | http://localhost:3000/supplier | Supplier Page |
| 4.  | http://localhost:3000/404      | 404 page      |
